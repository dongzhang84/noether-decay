# 学习作为诺特荷的衰减：一个耗散动力系统的视角

> 一份完整记录：从第一性原理推导，到全部定理，到 toy model 验证（含关键的否定结果），到物理立场与边界。
> 可直接作为 repo 主文档使用。

---

## 0. 一句话与口号

**学习就是对称性的破缺。** 在参数空间里，诺特荷的守恒意味着那个方向"没在学"（盲区），诺特荷的衰减才对应"在学"。学习的过程，就是不断把守恒的对称方向破缺掉、转化为衰减的学习方向的过程。

口号（内部定位）：**学习就是空间对称性的破缺。**

## 1. 方法论立场（为什么这样做）

我们刻意采用**物理学家的方法**，而不是机器学习常见的做法：

- **不做 cherry-picking**：不从复杂系统里挑两个小东西说它们等价（例如"BatchNorm ≈ RMSProp"那一类）。那样能写很多文章，但抓不住本质。
- **化简到最理想情形先立稳，再逐级加复杂度**：力学先从无力的惯性系（牛顿第一定律）出发，再加保守力、再加摩擦；我们也先从"损失为常数"的理想系统出发，再加势能、再加耗散。
- **学庞加莱**：三体问题解不出精确解时，庞加莱转而研究相空间的**定性结构与不变量**（不动点类型、指标、同宿结构、混沌），并由此证明了硬定理（不可积、回归定理）。我们也一样：训练轨迹解不出来，就抓**不变量**，而不是抓会乱变的精确轨迹或衰减率。
- **抓不变量，不抓速率**：庞加莱抓拓扑指标、Morse 抓临界点指标、耗散系统抓吸引子维数/李雅普诺夫指数——它们的共同特质是"在不可解的复杂性之上仍然稳定、可分类的不变量"。我们据此选择**对称破缺数（整数）**和**吸引子维数**作为目标，而不是诺特荷的衰减率（衰减率会被优化器锁死、会乱变，不可推广）。

## 2. 诺特定理与广义诺特定理

### 2.1 标准诺特定理

系统由拉格朗日量 $L(w,\dot w,t)$ 描述。考虑连续对称变换 $w\to Q(w,s)$，$Q(w,0)=w$，对称方向（生成元）

$$\delta w \equiv \left.\frac{\partial Q}{\partial s}\right|_{s=0}.$$

**诺特荷的第一性原理定义**（广义动量在对称方向上的投影）：

$$Q_N = \left\langle \frac{\partial L}{\partial \dot w},\ \delta w\right\rangle.$$

标准诺特定理：若 $L$ 在该变换下不变（对称未被破坏），则 $Q_N$ 守恒，$dQ_N/dt=0$。**前提是对称不被任何力破坏——这是保守系统的定理。**

### 2.2 守恒的局限

物理学家偏爱守恒：守恒最简单，且可作参照基准。但真实系统大多是耗散的、不守恒的，标准诺特定理对它们直接失效。物理把"守恒"研究得淋漓尽致，却把**"守恒如何被破坏"**留在角落——而学习恰恰是耗散过程，需要的正是这被冷落的一面。

### 2.3 广义诺特定理 = 平衡律（balance law）

对诺特荷直接求时间导数（乘积法则，纯文本写法）：

```
dQ_N/dt = ⟨ d/dt(∂L/∂ẇ), δw ⟩ + ⟨ ∂L/∂ẇ, d(δw)/dt ⟩
```

代入带广义力的运动方程 `d/dt(∂L/∂ẇ) − ∂L/∂w = F_力`，并利用对称性使保守部分相消，得到**平衡律**：

```
dQ_N/dt = ⟨ F_力, δw ⟩
```

即**诺特荷的变化率 = 作用在对称方向上的力做的功率**。守恒律是它的特例（力不做功时）。

> 文献现状（诚实标注）：耗散诺特定理在物理里有零散讨论（Galley 2013 的变量加倍；Delphenich 把守恒律推广为"非守恒流的平衡原理"），但相对冷门、引用不高、无权威教科书。守恒那一侧才是主流。这一现状本身印证了 2.2：物理对"衰减"研究不足，而这正是我们的切入点。

## 3. 把训练写成动力系统

从权重更新 `w ← w − η∇Loss` 出发，取连续极限得一阶梯度流 `ẇ = −∇Loss`。但欧拉–拉格朗日方程天生二阶——这是"裂缝"。补上惯性项 $m$（有限学习率的隐式加速）：

```
L = (1/2) m‖ẇ‖² − Loss(w) = 动能 T − 势能 V
```

- **loss = 势能 V**（参数运动的地形），不是总能量。
- 学习里给出连续对称、从而给出诺特荷的，**本质只来自重参数化冗余**（ReLU/重缩放对称）。两层线性网 $\hat y=W_2W_1x$ 的诺特荷是矩阵

```
C = W1 W1ᵀ − W2ᵀ W2
```

它在梯度流下严格守恒（即 Du–Arora 那条 balancedness），加入破坏对称的项后衰减。

## 4. 理想系统与三档层级

### 4.1 理想系统：损失为常数 —— 学习的"惯性系"

loss 处处平 → 无梯度、无力 → 所有诺特荷守恒。此时对称方向数 = 参数全维 $N$，对称性最完美（如均匀真空）。**但正因如此，这种情况什么都学不到。**

**核心反转：完美的守恒 = 完美的"没在学"。** 守恒不是成就，是学习的缺席。

### 4.2 三档理想化层级

| 档位 | loss | 力 | μ | 诺特荷行为 | 物理对应 |
|---|---|---|---|---|---|
| 档一 | 常数 | 无 | — | 守恒 | 惯性系（不学） |
| **档二（主战场）** | 有坡 | 各向同性 | 恒定 | 平滑指数衰减，率 μ/m | 惯性系+势+各向同性阻尼 |
| 档三 | 有坡 | 各向异性+噪声 | 时变 | 起伏漂移 | 现实 SGD |

现实 SGD 在档三，比档二多三样：**噪声**（minibatch 抽样）、**各向异性**（不同方向衰减不同，尤其配 Adam）、**μ 时变**（warmup、学习率衰减，至今靠经验）。**本文只在档二立定理**，档三是推广方向，不可直接外推。

## 5. 关于"摩擦"——我们的物理立场（重要，勿误解）

这一点最容易被纯优化背景的人误读，必须讲清：

**事实层面**：在 toy model 里，产生诺特荷衰减 `‖C‖ ~ e^(−2μt)` 的，形式上是 **weight decay**（位置项 −μw）。从纯优化角度，−μw = ∇(½μ‖w‖²)，可被视为一个保守正则项。我们已用代码核实：是 weight decay（位置阻尼）产生此衰减；速度阻尼 −μẇ 不产生（拟合衰减率≈0，甚至发散）。这个事实我们如实接受。

**立场层面（重点）**：我们是从**物理 / 耗散动力系统的视角**组织这套理论的。我们关心的不是"这一项在优化器代码里叫什么名字"，而是**它在动力学上扮演的角色**——它破坏对称、把诺特荷单调耗散掉、使相空间体积收缩。**这正是"摩擦/耗散"的功能性定义。** 无论它形式上写成位置项还是速度项，它在学习动力学中起的就是摩擦的作用。

因此本文同时坚持两件不矛盾的事：
1. **保留"摩擦/耗散"作为这一行为的物理称呼**（按其动力学角色命名）。
2. **如实承认**在 toy model 这条具体路径上，这个摩擦行为由 weight decay 实现（而非速度阻尼）；平衡律是更一般框架，本文走"正则/耗散破坏对称"这一条。

一句话：**我们说"摩擦"，指的是它在耗散动力学中的角色；它在实现上恰好是 weight decay。两者不矛盾，并行陈述，不用其一抹掉其二。**

## 6. 定理（逻辑链）

> 关于"定理"：以下多为**陈述精确、形式清楚、已用 toy model 验证**的命题；严格一般性证明仍需补足。按此标准诚实使用"定理"一词。

### 本质层

**T1（学习=诺特荷强制衰减）**：被学习的对称方向上，`dQ_N/dt = ⟨F_力, δw⟩`；且 `dQ_N/dt = 0 ⟺ 该方向不参与学习`。

**T1′（盲区分解，强版）**：参数空间 = 守恒子空间（盲区，学不到）⊕ 衰减子空间（在学）。**守恒量的数目 = 当前学习盲区的维度。** 学习 = 盲区缩小、衰减区扩大。

**T2（否定定理：衰减率与任务无关）**：各向同性恒定摩擦下，所有诺特荷以同一速率衰减，`Q(t)=Q(0)e^(−μt/m)`，速率只由优化器（μ/m）决定，与任务无关。解析即得，无需模拟。**它精确说明"衰减率"这个观测量本身不携带学习信息。**

### 庞加莱层

**T3（无封闭解析解）**：学习动力学一般无封闭解析解，这是"炼丹"（只能数值训练、反复试）的某种必然性来源。
> 注：原稿曾用 Liouville–Arnold "不可积"来包装这一条。这是 category error——L–A 可积性是哈密顿系统的概念，耗散系统不是哈密顿系统，套用不当；且非凸非线性系统无解析解本就平凡为真。故本条**降级为朴素陈述**，不强行拔高为"庞加莱式否定定理"。

**T4（任务信息只能从各向异性进入）**：诺特荷衰减携带任务信息，当且仅当作用在对称方向上的力是**各向异性**的（非欧度量或各向异性噪声）。各向同性力下，衰减与任务正交（这正是 T2 的来源）。

### 耗散层

**T5（相空间收缩到吸引子）**：学习使参数相空间体积单调收缩，运动落到比参数维数更低的吸引子上（刘维尔定理的耗散版：保守=体积守恒，耗散=体积收缩）。

**T6（收缩率 = 诺特荷衰减率之和 = 李雅普诺夫指数之和）**：三者相等。toy model 中 `‖C‖ ~ e^(−2μt)`（C 是二次量，率 2μ），数值与理论严格吻合。核查确认此衰减由 weight decay 产生。

**T7（正交分解：how ⊥ what）**：学习把参数自然分成两个正交部分：
```
任务内容（what）= 乘积 W2W1 的奇异谱   ⊥   内部组织（how）= 诺特荷 C
```
诺特荷度量"以什么方式学的（how / 层间组织 / 规范自由度）"，与"学了什么任务（what）"正交、独立。两网络学同一任务可有不同诺特荷；学不同任务可有相同诺特荷。

**T8（展望：收缩率 = 熵产生率）**：耗散系统收缩率 = 熵产生率，学习可视作熵增的不可逆过程（学习的热力学第二定律）。列为展望，正文不展开。

### 两个核心不变量（真正想推广的东西）

- **对称破缺数**（整数，拓扑不变量，庞加莱/Morse 式）：学习"激活"了多少个原本守恒（死）的方向 = 学习深度。
- **吸引子维数**（维数不变量，耗散系统式）：学习后系统实际活在几维 = 有效维度。
- 关系：**对称破缺数 + 吸引子维数 ≈ 参数总维数 N。**

衰减率会随系统复杂化乱变、不可推广；这两个整数/维数不变量有望从档二推广到档三。

## 7. Toy Model 与全部实验（含关键否定结果）

统一模型：两层线性网 `ŷ = W2 W1 x`，诺特荷 `C = W1 W1ᵀ − W2ᵀ W2`。一个模型贯穿全部定理。

### 7.1 主验证图（六格，对应 T1–T7）

- **T1**：纯 GD 下 Q 守恒（平线）；加摩擦 Q 衰减到 0。守恒=没学，衰减=在学。
- **T2**：三个不同秩任务，Q(t)/Q(0) 曲线完全重合 → 衰减率被 μ/m 锁死，看不到任务。
- **T3**：守恒荷上限 21 ≪ 自由度 72 → 守恒量远不够 → 无解析解。
- **T4**：各向同性摩擦下末态 ‖C‖ 对所有任务都是平的零；各向异性学习率下 ‖C‖ 开始随任务变。
- **T5+6**：‖C‖ 在对数轴是直线（指数收缩），与理论 e^(−2μt) 完美重合。最硬的一格。核查确认来自 weight decay。
- **T7**：诺特荷谱归零（任务无关）；乘积 W2W1 奇异谱随任务秩变化（rank=1→1 个奇异值，…，rank=6→6 个）。任务活在乘积谱，诺特荷与它正交。

（图：`eight_theorems_toymodel.png`）

### 7.2 关键的否定结果（不是失败，是想清方向的关键）

这些"死"出来的结果，恰恰逼出了正确的认识，必须记录：

1. **"用诺特荷衰减率区分任务" → 死**：换四个差异极大的任务（满秩/秩1/近恒等/大尺度），衰减率半衰期全是 ~1.15、完全一样。原因：`dQ/dt=−(μ/m)Q`，率只含 μ、不含任务。**这是 T2，解析即知，不必模拟。**

2. **"诺特荷衰减数 = 学习有效维度" → 被证伪**：换不同秩任务，整个诺特荷谱在摩擦下全部归零、与任务无关；噪声平衡设定下谱范数也几乎不随任务变。

3. **证伪的方式恰恰证明了 T7**：对照实验显示——诺特荷谱（任务无关）vs 乘积 W2W1 奇异谱（清楚随任务秩变化）。所以任务信息（what）在乘积谱里，诺特荷（how）与之正交。**原命题死，但它的死法直接立起了 T7。**

4. **物理根因**：诺特荷 `C=W1W1ᵀ−W2ᵀW2` 度量的是"层间不平衡"= 重缩放冗余方向 = 任务正交的死方向。所以它的衰减/平衡/谱本质上编码不了任务——这正是"守恒=死方向"的最强证据，也兑现了"诺特荷=how 的正交宏观量"的判断。

### 7.3 摩擦力核查（T6 机制）

分别跑两种力，看 `‖C‖` 衰减：
- weight decay（−μw）：衰减率 0.6008 ≈ 2μ。**吻合。**
- 速度阻尼（−μẇ）：衰减率 ≈ 0，不衰减。

结论：toy model 的 e^(−2μt) 由 weight decay 产生（见第 5 节立场说明）。

### 7.4 诚实的边界

- 数值硬验证：T1、T2、T5、T6、T7（T5+6 与理论严格吻合，最硬）。
- 示意性/待严格化：T3（数自由度）、T4（各向异性效应偏弱但确随任务变）。
- 展望：T8（热力学化）。
- 适用范围：结论在档二（各向同性、μ 恒定）严格成立；档三（现实 SGD）是推广方向，不可直接外推。

## 8. 与相关工作的关系

- **Tanaka–Kunin, "Noether's Learning Dynamics" (NeurIPS 2021)**：广义诺特方程 (NLD) + 动能对称破缺 (KSB)，落点 BatchNorm≈RMSProp。在真实算法上验证了一个具体等价。
- **Ziyin et al., "Parameter Symmetry and Noise Equilibrium of SGD" (NeurIPS 2024)**：SGD 噪声把诺特荷推到与初始化无关的固定点（noise equilibrium）。用 Itô 引理（随机微积分）证明，关键是二阶项 `Ċ = σ²Tr[Σ∇²C]`。

**我们的定位（中性表述）**：他们各自在特定对称+特定算法上验证了具体结论。我们提供的是一个不同的东西——**从最理想系统出发、用广义诺特定理给学习一个统一的耗散动力学视角，并提炼可推广的不变量（对称破缺数、吸引子维数）。** 这是统一/分类的尝试，与他们的具体结果不冲突、也不互相替代。（待档三的不变量真正站住后，再谈更强的定位。）
